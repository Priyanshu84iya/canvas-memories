## ğŸ“– About The Project

**Canvas & Memories** is a web-based drag-and-drop collage creator. Unlike static photo grids, this project mimics a real-life messy desk or corkboard. It allows users to scatter their photos freely, write handwritten notes, and immerse themselves in a creative environment.

It features a **Day/Night toggle** that not only changes the color palette but introduces **ambient rain sounds** for a cozy, focus-friendly atmosphere.
